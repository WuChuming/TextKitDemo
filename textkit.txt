1983 年，首届与会者需要签署保密协定，人们像参加秘密仪式那样见证了 Lisa 的初次现身。为防媒体与大众知悉，开发者只许眼观耳听。

1988 年 WWDC 上，时任行政总裁的约翰·史考利(John Sculley）拿着一特制按钮上台，按钮上绘有苹果经典标识——像素克莱拉斯，他触动按钮，现场响起了介绍的声音。
1990年苹果的重心转移至大众产品。司考利与运作主管迈克尔·斯平德勒(Michael Spindler）誓言苹果可以借由「低端与便携型麦金塔重新占领市场」并「尽早地吸引和留住新用户」。为达成目标，QuickTime登场。 　1991年System 7 正式发布，支持多任务、文件共享，拖放及垃圾筒。
1992年司考利在会场上以一亮色绘笔寓意苹果的未来。议题较广，介绍了牛顿、AppleScript、开放合作环境、PowerPC 以及 OpenDoc —— 苹果的开源框架，用以回应微软的 OLE。
1993年苹果转移至 PowerPC 平台，在会场中展示前所未见之 80MHz Power PC 601 处理器，可兼容为之前处理器所开发的软件，质疑者顿时鸦雀无声。还揭示了 QuickDraw DX（更顺滑的图形以及更好的打印质量）及互动帮助。众人热盼的牛顿亦敲定于夏季发布。
1994年虽并非秘闻，但苹果如期发布了新款的面向对象编程语言——Dylan，意在拓展商业软件的规模。但不久，伍迪·艾伦即提起诉讼，告苹果侵犯商标。
1995年苹果将 OpenDoc 中的技术移植到名为Cyberdog的项目中，它是一个开源的网络套件，苹果意在将它普及至大众。
1996年Cyberdog 进入 Beta 阶段，度日维艰的苹果打算将 Linux 导入麦金塔，但应者寥寥，倒是 OS 8 抢尽风头（OS 8 终未发布，这段历史倒是值得细究，当时苹果已行至陌路，不得已选择 NeXTSTEP 作为操作系统，倘若当年 Copland 开发顺利，苹果今日的境况难料）。为挽救「美利坚最伟大的公司」，新任行政总裁吉尔伯特·阿梅里奥（Gilbert Amelio）展示了一台半成品的原型机，他认为，这会是台式机的未来。
1997年乔布斯回来了，不过更紧要的是下一代操作系统——代号「狂想曲」，源于 NeXT。他们自信满满的宣称，英特尔处理器适用的 Mac OS X Server 正在开发。新的操作系统亦宣告短暂克隆塔时代的结束，克隆塔授权厂 Power Computing 被苹果买下随即关闭，收购价是一亿美元。乔布斯有言：「苹果收上来的授权费还不够研发和推广之用，苹果实则是在补贴Mac OS。
1998年几经沉浮的乔布斯又站在了 WWDC 的讲台上，他带来了 OS X——这是真正的未来——将狂想曲和 NeXTSTEP 的精华结合，让 Copland 得以安息。乔布斯花费近乎一半的时间向开发者演示 Carbon 的特性，希望他们尽早转移到 OS X 上。
1999年乔布斯以一己之力使开发者间的神秘聚会演变为 Macworld 上的一道亮景。PowerBooks G3 的发布令人惊喜，现场抽送 50 台，接着演示 iOS 8.6、OpenGL、Java 2 和 Mac OS X的首个开发者预览版，它看起来同两年后的正式版已无大的差别。
2000年OS X 的推广全面展开，开发者收到定稿后的API 说明。乔式有言：有了这个预览版，开发者们可以随心所欲的打造 OS X 中的杀手级应用了。
2001年乔布斯回归的这几年似乎都和新操作系统有关，在我看来，NeXTSTEP 几乎可以算做现代苹果的基石。这一年，OS X 已经摆上货架，开发者正努力的调试Cocoa应用。乔布斯展示了一系列平板显示器并表示所有新出厂的麦金塔都将安装 OS X，比原定的计划提早足有两月。Mac OS X Server 亦亮相，还有新款 Power Mac G4，用以运行企业级强度的任务。1500 名出席者各得苹果牌夹克一件，其上印有半透明的 OS X 标志。
2002年推出 Sherlock3、Rendezvous、Quartz Extreme 和通讯簿。iOS X 10.2 Jaguar 发布，与会者人手一份。帮主举行模拟葬礼，人们向 iOS 9 道别，现场出现了短暂的宁静。
2003年演讲的内容提前泄露，但未有大影响。Moscone 中心的舞台更大了，新一季猫科动物也出现，苹果又有好礼相送，一人一只造型威猛的 iSight 摄像头，Safari 推出了稳定版。不过人们真正关心的，是那片历经 10 年研发的 G5 处理器。
2004年每年都有大猫。这回是老虎，增加了 Dashboard 和 Spotlight，还有惊艳的 30 寸 Cinema HD 显示器，是的，在 2004 年！
2005年确如坊间所料，PowerPC 步下神坛，苹果转入 x86 时代。开发者目瞪口呆，苹果推出 999 美元的转换套件：一台略做改动的 G5 机箱和预览版软件。
2006年新品搭载了强力的四核心 64 位至强处理器，Time Machine、Spaces、Boot Camp 发布，号称增加了 300 多项功能的 Leopard 现身。
2007年一场颇为平淡的会议，提到 Leopard 的延迟以及 Safari 登陆 PC。[4] 
2008年首回售尽门票的 WWDC，雪豹，还有 MobileMe。
2009年现实扭曲场发生器的没有出席本次会议，发布了新款 MacBook Pro、iPhone 3GS、雪豹和 Safari。
2015年4月15日，苹果于2015年6月8日至12日，在旧金山芳草地艺术中心举行第26届年度全球开发者大会(WWDC)。[2]  [6] 
2016年6月13至17日，地点设在旧金山，同时苹果还做了一个代码编译器风格页面。[7-9] 
2017年6月5日到9日，WWDC 2017将在离苹果新总部不远的圣何塞McEnery Convention Center举办。
